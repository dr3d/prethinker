# Document Intake Samples

Small source/target fixtures for the document-to-logic compiler path.

These are not runtime scratch files. They preserve the Declaration and
Proclamation experiments as reusable calibration material for:

- hint-free predicate discovery;
- source-bound document compilation;
- optional fixture-owned predicate registries;
- expected Prolog-style comparisons;
- post-ingestion QA over the compiled KB.

The narrow registries in this folder are vocabulary/context fixtures only. They
do not contain source facts and they do not authorize writes. They are useful
for testing the Harbor-style finding that a curated working predicate surface can
beat both hint-free over-compression and giant-menu registry exposure.

Generated runs, traces, and exploratory outputs still belong under `tmp/`.

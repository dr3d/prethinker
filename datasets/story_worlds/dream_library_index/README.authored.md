# The Dream Library Index

## Premise
A community library's real incident log (pipe burst, missing book, heating failure, patron injury, inventory discrepancy) coincidentally mirrors plots in three recently acquired novels. The librarian files a report explicitly separating fiction from fact. Questions must not treat novel plots as real events, even when names, numbers, and details match.

## Intended Difficulty
Hard. Attacks **story level and reference containment**. The fictional novels describe events that superficially resemble real incidents — same book title, same volunteer name, same discrepancy count, same librarian name. The system must not treat fictional events as facts about the real library.

## Domain
Library administration / incident reporting / fiction-fact boundary

# The Orchard Inheritance Game

## Premise
Over four generations, an orchard's twelve named fruit trees pass through a family via inheritance, marriage gifts, debt pledges, sales, verbal loans, and one contested will. The current state depends on tracking every transaction type distinctly.

## Intended Difficulty
Hard. Attacks **possession vs ownership vs inheritance** — physically having something, legally owning it, having it pledged, having it borrowed, and having it gifted without formal transfer are all different states.

## Domain
Family property / succession / multi-generational estate / probate

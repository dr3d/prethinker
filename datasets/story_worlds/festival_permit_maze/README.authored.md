# The Festival Permit Maze

## Premise
A 10-day autumn festival requires five different permit types, each with different issuing authorities, timelines, extension rules, violation consequences, and appeal procedures. By day 7, multiple permits are suspended, restricted, or expiring, creating a complex web of overlapping regulatory states.

## Intended Difficulty
Hard. Attacks **rule activation and deadline families**. Each permit type has its own clock, its own violation triggers, its own suspension mechanics, and its own renewal/extension rules. Questions require tracking five parallel regulatory timelines simultaneously.

## Domain
Municipal event regulation / multi-permit compliance / deadline arithmetic

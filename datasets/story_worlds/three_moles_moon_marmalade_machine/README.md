# The Three Moles and the Moon-Marmalade Machine

This fixture is a source-local story-world frontier project for Prethinker.

It is intentionally adjacent to the Otters/Goldilocks-shaped story family but
has its own source-local characters, objects, magic mechanics, harms,
restitution, and final-state moral. It should be useful for checking whether
context engineering generalizes without inbreeding on one story.

## Files

- `story.md`: source story to compile.
- `source.md`: preserved copy of the story under the generic source name.
- `qa_source.md`: original supplied 40-question QA table with likely mistakes.
- `qa.md`: script-compatible numbered QA file for post-ingestion QA.
- `progress_journal.md`: running mini-journal of benchmark attempts and lessons.
- `progress_metrics.jsonl`: append-only metrics rows for progress graphs.

## Evidence-Lane Discipline

This fixture intentionally has no `gold_kb.pl`, no `ontology_registry.json`, and
no strategy/intake-plan notes. Do not add those as part of fixture admission.

Cold runs should use only `story.md` as the source document. `qa.md` and
`qa_source.md` exist for post-ingestion scoring, not source-compilation context.

No benchmark run has been recorded yet.

## Why It Matters

The fixture pressures:

- source-local identity instead of Otters/Goldilocks prior contamination;
- repeated triads across cups, caps, carts, jars, and fog-leaves;
- literal-but-magical objects such as beetle-keys, spare minutes, and pickled
  thunder;
- speech versus narrator truth, especially Mina's excuses and the speaking cap;
- bodily/final-state updates around humming, swallowed keys, and memory;
- causality and restitution after object misuse.

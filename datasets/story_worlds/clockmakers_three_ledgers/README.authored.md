# The Clockmaker's Three Ledgers

## Premise

A deceased clockmaker left three handwritten ledgers covering his repair shop's accounts, client records, and parts inventory. The ledgers overlap in time but disagree on dates, ownership of consigned pieces, repair status, and debts. Marginal notes in different ink correct some entries but introduce new contradictions. An estate executor must reconstruct the true state of affairs.

## Intended Difficulty

Hard. The fixture attacks **correction provenance and stale-vs-current state**. Three sources disagree; later corrections fix some problems but not others; one correction is itself wrong; and the current state of several items depends on which ledger's corrections are authoritative.

## Domain

Estate administration / small business records / conflicting documentation

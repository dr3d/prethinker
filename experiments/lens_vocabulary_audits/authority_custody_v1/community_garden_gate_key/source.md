# Community Garden Gate Key Packet

The garden council opened packet GK-6 for the north gate key after a lock replacement.

The official key register KR-6 was filed by secretary Ona Bell on 2026-09-01. KR-6 names gate coordinator Jae Park as custody holder and marks the key as restricted.

Staff note SN-6 was written by volunteer lead Emi Frost on 2026-09-02. The note recommended lending the key to plot captains during harvest week, but it was only a staff note.

Draft recommendation DR-6 was prepared by access aide Hugo Lin on 2026-09-03. DR-6 proposed weekly key checkout for all members, but it remained draft pending a council vote.

Garden access rule GA-2 governs gate keys. GA-2 says restricted keys stay with the gate coordinator unless the council votes or a court order changes access.

On 2026-09-04, the council voted to allow temporary checkout only to plot captains and to keep custody with Jae Park.

Court order CO-6 was issued by Judge Silva on 2026-09-05. CO-6 required emergency access for fire inspection but did not authorize member checkout.

The council chair issued controlling finding CF-6 on 2026-09-06. CF-6 says Jae Park remains custody holder, the council vote controls temporary checkout, and CO-6 controls fire inspection access only.

A copied shed poster SP-6 says "all members may check out the key weekly." SP-6 is noncontrolling because it copied DR-6 and omitted the council vote and CF-6.

Access to the north gate key is controlled by gate coordinator Jae Park.

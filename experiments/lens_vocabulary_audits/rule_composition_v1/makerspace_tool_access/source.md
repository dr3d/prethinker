# Rule Composition Probe: Makerspace Tool Access

The North Pier Makerspace adopted Tool Access Notice TA-4 for weekend use of
the laser cutter.

The base rule says a member may reserve the cutter only after completing two
supervised practice sessions. The activation condition is a weekend reservation
request for the laser cutter.

There is an exception for members who hold an instructor endorsement: if an
instructor signs the endorsement, the two-session practice requirement is
waived for that member's next weekend reservation only.

The safety hold rule overrides ordinary reservation approval whenever the
cutter's exhaust alarm is active. If both the endorsement exception and the
safety hold rule could apply, the safety hold rule takes precedence.

The endorsement exception expires on 2026-08-31. If the weekend roster tool is
offline, the fallback rule is to use the paper sign-up sheet at the front desk.

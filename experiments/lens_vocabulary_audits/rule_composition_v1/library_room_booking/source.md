# Rule Composition Probe: Library Room Booking

The West Hill Library issued Room Booking Memo RB-12.

The base rule says study rooms may be booked for two hours at a time. The rule
activates when a group asks for a room during public service hours.

An eligibility condition says the requester must hold a current library card.
The threshold for a large-group booking is eight or more attendees.

Large-group bookings require staff approval before confirmation. The exception
is a public exam-prep session sponsored by a school: if the school sponsor is
named on the request, the staff-approval requirement is replaced by same-day
notification to the branch manager.

Emergency closure notices override every room-booking rule. If an emergency
closure notice conflicts with the school-sponsored exception, the emergency
closure notice has precedence.

RB-12 is valid until 2026-12-31. If the booking calendar is unavailable, the
fallback rule is to record requests in the blue room ledger.

# Layered Multi-Subject Authority Pair

Boundary class: `layered_multi_subject_authority_coverage`

This probe tests whether the compile and QA layers preserve multi-subject
authority when the authority is mediated through a correspondence note,
decision packet, or source reference rather than stated as a single direct row.

The fixture-free geometry is:

- a surface table or summary names governed subjects;
- a separate source layer identifies the authority document, actor, and date;
- the authority document governs multiple subjects;
- a nearby non-authority note or older source is present;
- questions ask for both governed subjects and the source layer that supports
  one governed subject.

No repair should name this probe, its identifiers, or its domains.

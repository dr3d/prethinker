# Boundary Probe: Counterfactual Arithmetic Unstated Pair

## Probe A: Reserve Calculation Memo

The active reserve count for register G-21 is 146 units.

Change C-8 proposed adding 17 standby units to register G-21. C-8 remained
pending at the close of the memo, so the 17 standby units are excluded from the
active reserve count.

For the optional planning view, calculate the count by taking the active
reserve count and adding the 17 standby units from C-8. The memo does not print
the resulting planning count.

## Probe B: Materials Rework Sheet

The approved release count for lot T-63 is 312 parts.

Revision V-2 proposed subtracting 24 parts that were marked for rework. V-2 was
not approved before release, so those 24 parts were not subtracted from the
approved release count.

For the alternate rework view, calculate the count by taking the approved
release count and subtracting the 24 rework-marked parts from V-2. The sheet
does not print the resulting rework count.

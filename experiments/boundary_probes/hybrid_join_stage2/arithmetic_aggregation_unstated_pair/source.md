# Boundary Probe: Arithmetic Aggregation Unstated Pair

## Probe A: Equipment Cost Review

Work order WO-74 has four approved cost lines:

| Line | Cost class | Amount |
| --- | --- | ---: |
| L-1 | parts | 184 |
| L-2 | labor | 96 |
| L-3 | testing | 42 |
| L-4 | disposal | 18 |

Line L-5 is a proposed rush-handling surcharge for 27. L-5 was not approved
before closeout and is excluded from the approved work-order total.

For the approved-total view, calculate the total by summing only approved cost
lines L-1 through L-4. The review does not print the resulting approved total.

For the proposed-inclusive view, calculate the total by adding proposed line
L-5 to the approved-total view. The review does not print the resulting
proposed-inclusive total.

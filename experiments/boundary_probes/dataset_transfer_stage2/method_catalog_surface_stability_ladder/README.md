# Method Catalog Surface Stability Ladder

This probe tests whether compile preserves several answer-bearing surfaces at
once when a passage describes a method catalog.

The pressure is simultaneous preservation:

- a frame purpose for a work process,
- named methods that serve that purpose,
- method-specific measurements or observed properties,
- operating conditions,
- context or location phrases.

The fixture is not repair permission by itself. It is a stability probe for a
surface that already showed transfer in direct and dense purpose-frame forms,
but became unstable when a recompile gained purpose rows while losing detail or
location resolution.

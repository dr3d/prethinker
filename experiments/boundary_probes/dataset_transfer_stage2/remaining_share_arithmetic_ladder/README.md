# Remaining-Share Arithmetic Ladder

This probe tests residual-share questions: a source gives a partition of a
whole, one named share, and enough structure to identify the remaining share,
recipient, or amount.

This is narrower than general arithmetic aggregation. The target is whether
the system can keep the partition view, stated share, whole, and residual
recipient together without inventing source facts.

The ladder includes:

- residual recipient from a named remaining-share row
- residual fraction when complementary fractions are printed
- residual amount when whole and first share are printed
- residual amount with a named residual recipient
- distractor partitions that should not enter the calculation

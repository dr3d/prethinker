# Printed Relative Distance Ladder

Purpose: isolate temporal-distance surfaces where the source states an event,
an endpoint, and a printed relative distance such as "two years before" or
"three months after." The answer-bearing coordinate is the printed distance,
not only the two endpoint dates or a before/after relation.

Forbidden fix: do not add selectors, helpers, predicates, or prompts for this
packet's names, events, years, or answer strings. A useful repair must be a
generic compile/query principle:

- preserve printed relative distances as queryable surfaces;
- keep endpoint dates and before/after ordering as separate evidence;
- do not require arithmetic if the source already prints the distance;
- do not collapse the distance into a long source-text atom only.

# Causal-Chain End-State Ladder Source

The safety recall led to the closure order. The closure order ended the pilot
expansion. A later archive memo only documented the closure order and did not
cause it.

The funding freeze caused the permit cancellation. The permit cancellation
ended the river survey. The public notice announced the cancellation after it
was already final.

The storm warning triggered the harbor shutdown. The harbor shutdown ended the
night ferry service. The maintenance checklist was completed after the shutdown.

The court ruling led to the license withdrawal. The license withdrawal ended
the market trial. The agency bulletin summarized the ruling but did not withdraw
the license.

The software fault caused the emergency rollback. The emergency rollback ended
the beta release. A customer message described the rollback but did not cause
the release to end.

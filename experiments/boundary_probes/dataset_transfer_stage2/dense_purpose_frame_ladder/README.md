# Dense Purpose Frame Ladder

This probe tests purpose recovery when the source states a paragraph-level task
and then introduces methods or instruments that serve that task.

The key pressure is not direct "tool is used for X" phrasing. It is frame
inheritance:

- A task is stated for a group or process.
- Several methods/instruments are listed as ways to perform that task.
- Later sentences describe lower-level measurements for each method.
- QA asks what a specific method/instrument is used for.

The fixture should not authorize a repair by itself. It is a density probe for
purpose surfaces that are already interior in direct phrasing.

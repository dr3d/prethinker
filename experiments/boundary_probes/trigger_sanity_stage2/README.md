# Trigger Sanity Stage 2

Focused follow-up probes from BH-025.

These probes isolate two pressures without reusing the stage-1 surface terms:

- `canonical_alias_register`: the same entity appears as a bare tag and as a
  noun-prefixed tag. The noun phrase must not create duplicate entities.
- `multilingual_source_fidelity_card`: non-English source text states final
  counts, excluded pending items, and a withdrawn estimate. Those distinctions
  should become queryable coordinates or source-addressable evidence without
  language-specific harness whitelists.

Forbidden fix: do not add fixture names, local identifiers, answer strings,
question ids, row ids, or language-specific word lists to harness architecture.

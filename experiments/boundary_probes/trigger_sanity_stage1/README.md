# Trigger Sanity Stage 1

Fresh unseen probes for auditing helper trigger generality.

These are not replacement benchmarks. They are small pressure fixtures designed
to ask whether recent helpers and classifiers depend on current-corpus naming or
document shape:

- `sectionless_resolution_notebook`: scoped counts without numbered section
  hierarchy.
- `state_word_register`: status-like facts using state/condition vocabulary
  rather than explicit status wording.
- `spanish_count_statement`: explicit compact counts in a Spanish administrative
  note with English QA.

Forbidden fix: do not add fixture names, local identifiers, answer strings,
language-specific whitelists, or row ids to harness architecture.

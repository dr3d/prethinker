## Pending Count Decoy

A pending addendum proposed adding 2 provisional units to the affected set. The
addendum was not adopted before close of packet, so the 2 provisional units are
excluded from the closing stated total.

## Withdrawn Estimate Decoy

An analyst note estimated that the affected set might reach 15 units after a
future reconciliation. The analyst withdrew that estimate on August 20, 2026.
The withdrawn estimate is not the closing stated total.

## Closure Rule

When a question asks for the stated total, use the compact count statement. When
a question asks for the itemized register count, count only listed register
items. Do not include pending or withdrawn estimates in either closing count.

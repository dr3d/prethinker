Today's Research Headline

Semantic Parallax: One Compile Is A Viewpoint.

The newest gain is multi-pass semantic compilation: backbone, support/source, temporal/status, and rule lenses can each propose a different view of the same source. The mapper admits each view independently, and deterministic union accumulates only admitted clauses.

Glass Tide is now the live frontier: broad compiles preserve charter rules as records, while separate rule lenses, body-fact lenses, helper substrates, and deterministic union are producing probe-gated rule surfaces across role joins, thresholds, exceptions, temporal windows, and council overrides.

Prethinker Documentation Hub

Prethinker is a governed write layer between natural language and a deterministic knowledge base. The LLM proposes; it does not get to commit.

Built to catch the failure mode where an LLM sounds right, writes the wrong thing, and narrates the corruption confidently.

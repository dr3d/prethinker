## What Is Intentionally Gone

The public docs should not point new readers at:

- lab automation plans;
- public benchmarking or publishing plans;
- generated run reports;
- long worksheet diaries;
- old prompt snapshots;
- retired compatibility-adapter archaeology;
- dated product-horizon notes.

Those materials are preserved by Git history or the local cold archive. If an
old artifact contains a lesson that still matters, summarize the lesson in one
of the current docs above instead of resurfacing the old artifact.

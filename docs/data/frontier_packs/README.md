# Frontier Packs

These JSON packs are small, tracked pressure fixtures for Semantic IR and
router/runtime frontier checks. They are not generated reports and they are not
gold KBs.

Current packs:

- `semantic_ir_cross_turn_frontier_pack_v1.json` - cross-turn memory,
  correction, and authority-boundary probes.
- `semantic_ir_lava_pack_v2.json` - early Lava semantic-pressure pack.
- `semantic_ir_lava_pack_v3.json` - expanded Lava semantic-pressure pack.
- `semantic_ir_lava_pack_v4.json` - source-record alignment pressure pack.
- `semantic_ir_lava_pack_v5.json` - current Lava pack used by recent
  admission-safety and semantic-clean checks.

Use the public docs hub for orientation and
[PROJECT_STATE.md](https://github.com/dr3d/prethinker/blob/main/PROJECT_STATE.md)
for current verification claims.
